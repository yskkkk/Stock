iPhone에 앱 설치

App Store 없이 Safari에서 「홈 화면에 추가」하면 웹과 같은 전체 화면 앱으로 쓸 수 있습니다.

이미 설치된 상태입니다.

홈 화면의 「종목 대시보드」 아이콘을 눌러 실행하세요.

앱 열기
카카오·인스타 등 앱 안 브라우저에서는 설치할 수 없습니다.

아래 버튼으로 주소를 복사한 뒤 Safari를 열고 주소창에 붙여넣으세요.

복사했습니다. Safari → 주소창에 붙여넣기

설치 방법 (Safari)

  1. Safari 하단 공유 버튼(□ 위 화살표)을 누릅니다.
  2. 메뉴를 아래로 스크롤해 홈 화면에 추가를 선택합니다.
  3. 이름 확인 후 오른쪽 위 추가를 누릅니다.
대시보드 열기 (설치 전 미리보기)

설치 후 홈 화면 아이콘으로 실행하면 주소창 없이 앱처럼 동작합니다. PC와 같은 Wi‑Fi의 서버 주소로 접속하면 됩니다.

네이티브 앱 (Swift · IPA)

갤럭시 APK와 같이 서버에 올린 웹 주소를 고정 URL로 여는 iPhone 앱입니다. Mac에서 npm run ipa:build 후 설치할 수 있습니다.

앱 설치 (기업·개발 배포) IPA 파일 다운로드

아직 IPA가 없습니다. Mac + Xcode + Apple Developer Team ID로 빌드하세요.

설치 후 설정 → 일반 → VPN 및 기기 관리에서 개발자 앱을 신뢰해야 실행됩니다. 등록된 기기(Ad Hoc/Development)에서만 동작합니다.
