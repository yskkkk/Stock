종목 대시보드 모바일 앱

웹과 동일한 화면·기능입니다. Android(갤럭시 등)와 iPhone용 앱을 설치할 수 있습니다.

Android (갤럭시 · 기타)

APK 다운로드 웹에서 열기

서버에 APK가 없으면 PC에서 npm run apk:build 실행 후 다시 시도하세요. 출처를 신뢰할 때만 설치하세요.

iPhone (iOS)

iPhone에 설치 (홈 화면) 웹에서 열기

Safari에서 공유 → 홈 화면에 추가로 설치합니다. App Store APK처럼 한 번에 받을 수 없어 이 방식이 iPhone 공식 설치 방법입니다.

앱 동작

APK는 빌드 시 정해 둔 Stock 서버 주소(예: 이 사이트와 같은 HTTPS)를 그대로 WebView로 엽니다. 웹에서 쓰는 화면·기능과 동일합니다. 서버 관리자가 npm run apk:build 전에 .env에 CAPACITOR_SERVER_URL을 넣어 두면 사용자는 주소 입력 없이 바로 사용합니다.
